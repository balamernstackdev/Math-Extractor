## Usage in Python

```python
from utils.html_entity_utils import decode_html_entity

# Decode entities
alpha = decode_html_entity("&alpha;")  # "α"
integral = decode_html_entity("&#x222B;")  # "∫"
```

For the complete list of 937 entities, see `entity_reference.json` or use the utility functions in `html_entity_utils.py`.
